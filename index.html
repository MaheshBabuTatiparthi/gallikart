Example

- https://localhost/generalstudies/restapi.php?api=books
- https://localhost/generalstudies/restapi.php?api=books&orderby=book_id
- https://localhost/generalstudies/restapi.php?api=books&orderby=random
- https://localhost/generalstudies/restapi.php?api=books&orderby=book_id&sort=desc



Retrieve a Book 

GET https://localhost/generalstudies/restapi.php?api=books&book-id=<book-id>

Parameters

ParameterValueDescriptionData Type
book-idUnique identifier for the object and use random for random IDsInteger

Example

- https://localhost/generalstudies/restapi.php?api=books&book-id=1
- https://localhost/generalstudies/restapi.php?api=books&book-id=2
- https://localhost/generalstudies/restapi.php?api=books&book-id=random
